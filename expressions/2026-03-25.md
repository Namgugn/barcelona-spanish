🇪🇸 오늘의 바르셀로나 스페인어

📌 상황: 카페나 음식점에서 포장 주문할 때

💬 스페인어: Para llevar, por favor.
🔊 발음: 빠라 예바르, 뽀르 파보르
🇰🇷 한국어: 포장해 주세요.

🔍 단어 분석:
Para → ~을 위해, ~하기 위해
llevar → 가져가다, 들고 가다
por favor → 부탁합니다, 주세요

✏️ 예시 대화:
- "Un café con leche, para llevar, por favor." (카페라떼 하나 포장해 주세요.)
- "Dos bocadillos para llevar." (샌드위치 두 개 포장이요.)

💡 팁: 바르셀로나에서는 테이크아웃 시 일회용 컵에 추가 요금을 부과하는 곳도 있습니다. 환경을 위해 텀블러를 가져가면 좋아요!
