# 2026-03-06

🇪🇸 오늘의 바르셀로나 스페인어

📌 상황: 카페, 식당, 호텔에서 와이파이를 물어볼 때

💬 스페인어: ¿Hay WiFi gratis?
🔊 발음: 아이 와이파이 그라티스?
🇰🇷 한국어: 무료 와이파이 있나요?

🔍 단어 분석:
¿Hay? → 있나요? (동사 haber의 3인칭 단수형)
WiFi → 와이파이 (영어 차용어)
gratis → 무료의, 공짜의

✏️ 예시 대화:
손님: ¿Hay WiFi gratis?
직원: Sí, la contraseña es "Barcelona2026". (네, 비밀번호는 "Barcelona2026"이에요.)

💡 팁: 비밀번호는 "¿Cuál es la contraseña del WiFi?" (꾸알 에스 라 꼰뜨라세냐 델 와이파이?)라고 물으면 됩니다. 바르셀로나 대부분의 카페와 식당에서는 무료 WiFi를 제공하지만, 비밀번호가 필요한 경우가 많아요!
