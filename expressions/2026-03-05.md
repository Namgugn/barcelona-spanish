# 2026-03-05

🇪🇸 오늘의 바르셀로나 스페인어

📌 상황: 식당이나 투어 예약할 때

💬 스페인어: ¿Puedo hacer una reserva para dos personas?
🔊 발음: 뿌에도 아세르 우나 레세르바 빠라 도스 뻬르소나스?
🇰🇷 한국어: 두 명 예약할 수 있나요?

🔍 단어 분석:
¿Puedo → 할 수 있나요? (poder 동사 1인칭)
hacer → 하다, 만들다
una → 하나의 (여성형 부정관사)
reserva → 예약
para → ~을 위해
dos → 둘, 2
personas → 사람들

✏️ 예시 대화:
손님: "¿Puedo hacer una reserva para dos personas?"
직원: "Sí, ¿para qué día y a qué hora?" (네, 언제 몇 시로 하시겠어요?)

💡 팁: 바르셀로나의 인기 레스토랑은 미리 예약이 필수! 예약 시 "para mañana a las ocho de la noche" (내일 저녁 8시로) 같이 날짜와 시간을 함께 말하세요.
