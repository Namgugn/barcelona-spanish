# 2026-03-21

🇪🇸 오늘의 바르셀로나 스페인어

📌 상황: 레스토랑에서 자리를 요청할 때

💬 스페인어: ¿Tienen mesa para dos?
🔊 발음: 띠에넨 메사 빠라 도스?
🇰🇷 한국어: 2명 자리 있나요?

🔍 단어 분석:
¿ → 의문문 시작 기호
Tienen → 가지다, 있다 (tener 동사의 3인칭 복수형)
mesa → 테이블, 식탁
para → ~을 위한
dos → 둘, 2
? → 의문문 종료 기호

✏️ 예시 대화:
당신: ¿Tienen mesa para dos?
직원: Sí, por supuesto. Síganme. (네, 물론이죠. 따라오세요.)

💡 팁: 바르셀로나에서는 점심 시간(14:00-16:00)과 저녁 시간(21:00-23:00)이 늦습니다. 인기 있는 레스토랑은 예약 없이 가면 자리가 없을 수 있으니 "¿Tienen mesa para dos?"라고 먼저 물어보는 게 좋습니다!
